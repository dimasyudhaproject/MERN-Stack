### 🏠 [Homepage](https://github.com/dimasyudhaproject/dimasyudha)

### ✨ [Demo](https://prosehat-dimasyudha.herokuapp.com/)

## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run dev
```
